# Visitor Pass Management System

A web-based Visitor Pass Management System designed to streamline visitor registration, approval, entry, and exit management within an organization or campus.

## Features (Planned)

- Visitor Registration
- Pass Generation
- Check-In / Check-Out Tracking
- Employee Approval Workflow
- Visitor Records Management
- Admin Dashboard & Analytics

## Tech Stack

- React
- Vite
- JavaScript
- Node.js (Planned)
- Express.js (Planned)
- MongoDB (Planned)
